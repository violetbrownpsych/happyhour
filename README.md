# pours
Twin Cities happy hour website

Here are the files you'll see:
1. index.html: The index file with the full html code for the site
2. happyhours.csv: The csv file with the happy hour data
3. manifest.json: A web manifest file in the repo root that describes the app (name, colors, icons). Cloudflare Pages serves it automatically.
4. sw.js: A service worker, which is just a small JS file that makes the app work offline.
5. tos.html: Terms of Service page. Cloudflare Pages serves it automatically.
